Meu primeiro Spring boot

MVC é o tipo de organização usado para classificar as pastas onde:

M - model = é onde gerencia os dados e a lógica e também é a parte onde o banco de dados se conecta com o que esta sendo produzido.
V - view = de maneira simples, é onde fica toda parte visual do programa como telas e as imagens.
C - Controller = é a parte do código onde armazena o que o programa pede para o usuario pela view.
